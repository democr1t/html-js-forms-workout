Forms workout

Your goal is to write 4 forms, each for different endpoint, that are handling form submissions with JavaScript and fetch.

Requirements

  • Use semantic HTML (input type="number" for numbers, etc.);
  • Use client-side form validation (at least form attributes like pattern, required);
  • Use special form attributes (enterkeyhint, autocomplete etc.);
  • Have styles/distinguish for valid/invalid/required states, import separate .css file;
  • Send form requests with fetch, import separate .js file;
  • Display errors (if any) for each input next to it;
  • Display result for successfull request under form.

Here is a code snippet for sending POST reques via form's submit event: 

const form = document.getElementById('search');
form.addEventListener('submit', (evt) => {
    const data = new FormData(evt.target);
    const obj = {};
    for (const key of data.keys()) {
        obj[key] = data.get(key);
    }
    const body = JSON.stringify(obj);

    fetch(form.action, {
        method: form.method,
        headers: {
            'Content-Type': 'application/json'
        },
        body
    });
});

Forms

Implement following forms:

  1. Search form;
  2. Sign-up form;
  3. Payment form;
  4. Delivery form.
Base path for server is http://localhost:3000/.
